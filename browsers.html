Resource Allocation

Browser and OS images use separate resource allocations configured in SiteSettings:

  • Browser images — browser_vcpu and browser_memory_gb
  • OS images (Kali) — os_vcpu and os_memory_gb

OS images use higher defaults because they run a full desktop environment.

Selecting Images at Setup

During setup.sh, you are prompted to select which images to install. Your selection is written to terraform/terraform.tfvars under docker_images. Only selected images will have ECS task definitions created and Docker images built and pushed to ECR.

To add images later, update docker_images in terraform.tfvars and re-run terraform apply, then rebuild with build_browsers.sh.
