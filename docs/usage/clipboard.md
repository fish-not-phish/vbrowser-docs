## Copy (Session → Device)

:::info

You should be able to copy directly from the session to your local machine without any special workflow. However, if you run into issues, follow the below workflow.

:::

1. Copy content inside the session.
2. Paste directly on your device.

![Clipboard Copy](./assets/clipboard.jpg)

:::warning
Some browsers require use of the clipboard text box.
:::

## Paste (Device → Session)

:::info

You should be able to paste directly into your sessions without any special workflow. However, if you run into issues, follow the below workflow.

:::

1. Copy content on your device.
2. Open the left-side menu.
3. Select **Clipboard**.
4. Paste into the text box.

![Clipboard Paste](./assets/clipboard.jpg)
